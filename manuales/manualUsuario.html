Manual de Usuario - TardeoAketza

La aplicación "TardeoAketza" permite configurar una URL y una palabra clave para realizar búsquedas periódicas en una página web. La tarea de búsqueda se ejecuta en segundo plano y se puede iniciar o detener en cualquier momento a través de los botones.

Requisitos

  • Permisos necesarios: Permiso para notificaciones (para Android 13 o superior).
  • Conexión a Internet: Se requiere conexión a Internet para realizar las búsquedas en la web.

Pantalla Principal

La pantalla principal de la aplicación contiene los siguientes elementos:

  • Campo de texto para la URL: Introduzca la URL de la página web donde desea realizar la búsqueda.
  • Campo de texto para la palabra clave: Introduzca la palabra clave que desea buscar en la URL indicada.
  • Botón "Play": Inicia la tarea periódica para buscar la palabra clave en la URL configurada. La tarea se ejecutará cada 15 minutos en segundo plano.
  • Botón "Stop": Detiene la tarea de búsqueda periódica.

Cómo Usar la Aplicación

  1. Configurar la URL y la palabra clave: Abra la aplicación y vea los campos de texto para la URL y la palabra clave. Ingrese la URL de la página web y la palabra clave que está buscando.
  2. Iniciar la búsqueda: Haga clic en el botón "Play" para comenzar la tarea de búsqueda periódica. Asegúrese de que ambos campos estén correctamente configurados. La tarea se ejecutará cada 15 minutos en segundo plano.
  3. Detener la búsqueda: Haga clic en el botón "Stop" para cancelar la tarea de búsqueda. Esto detendrá cualquier tarea programada y finalizará cualquier trabajo en ejecución.

Permisos

La aplicación solicitará el permiso de notificaciones en dispositivos con Android 13 o superior. Este permiso es necesario para enviar alertas de notificación si la búsqueda se completa correctamente.
